# Data

This folder contains synthetic data for a product analytics case. No real or sensitive data is used.

## Files

- `sample_users.csv`: synthetic user profile and acquisition data.
- `sample_events.csv`: product event stream with event names and timestamps.
- `sample_marketplace_actions.csv`: simplified marketplace opportunities and confirmation lifecycle.

## Notes

The sample CSV files are intentionally small so they can be reviewed directly on GitHub. The script `scripts/generate_product_events.py` generates a larger dataset under `data/generated/`.
